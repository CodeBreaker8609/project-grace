# Grace — Support Dashboard App

Grace is a simple, all-in-one dashboard designed to help users manage pain, stress, and financial concerns in a calm and structured way.

This project focuses on turning overwhelming situations into small, manageable steps through thoughtful UI and real-world logic.

---

## 🌿 Overview

Grace helps users:

- Track physical discomfort and receive supportive suggestions
- Process worries and decide what can (and cannot) be controlled
- Organize financial concerns into clear priorities
- Receive daily encouragement
- Reflect on small wins and positive moments

The goal is not to solve everything at once — but to help users slow down and take one step forward.

---

## ✨ Features

### 🧠 Body Check
- Pain level tracking (1–10)
- Area selection and notes
- Conditional support messaging
- Real-time UI updates

### 💭 Clear Your Mind
- Worry input with control-based decision making
- Action vs. release logic
- Personalized response messages

### 💰 Financial Focus
- Add financial concerns with priority levels
- Optional cost tracking
- Dynamic list rendering
- Remove items
- Summary of top priority

### 🌤 Daily Grace
- Random encouragement generator
- Updates dashboard summary
- Persists last message

### 🌱 Reflection
- Log something positive
- Track a small win
- Encourages daily mindset shifts

### 💾 Data Persistence
- Uses `localStorage`
- Saves all user input
- Restores state on page reload

### 🎨 UX Enhancements
- Styled validation (success, warning, error)
- Smooth message transitions
- Clear feedback for every interaction
- Reset / Clear All Data functionality
